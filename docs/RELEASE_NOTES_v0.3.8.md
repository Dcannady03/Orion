# Orion v0.3.8 — Homecoming

Orion Home turns startup into a compact personal command center rather than a technical boot banner.

## Highlights

- New `home` command refreshes the command center at any time.
- Startup now shows date, time, location, weather, calendar, workspace, AI, and application cards through the existing provider-neutral briefing system.
- AI Control Center receives a clearer panel layout with online status, capability checks, model size, parameter count, and context window.
- Stale knowledge indexes from another workspace are no longer injected into AI prompts.
- Existing `briefing`, model switching, profiles, and provider commands remain compatible.

## Verification

- 117 automated tests pass.

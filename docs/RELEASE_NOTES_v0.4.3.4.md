# Orion v0.4.3.4 — Signal: Channel Access

Orion can now operate as a shared assistant inside approved Discord channels.

## Commands

```text
discord bot access channel
discord bot access private
discord bot status
```

Channel-wide mode allows any non-bot member who can post in an allowed channel to mention Orion and receive a response. Direct messages remain limited to approved user IDs. Allowed-channel and required-role restrictions continue to apply.

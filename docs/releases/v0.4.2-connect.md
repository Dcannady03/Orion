# Orion v0.4.2 — Connect

> Historical note: Courier v0.6.0 replaces this Gmail-specific implementation with
> provider-neutral read-only Gmail and Microsoft Mail. The legacy direct-send path is
> disabled until Phase B provides persisted immutable outbound approvals.

Orion Connect introduces a unified communications center.

## Included
- Gmail OAuth setup through a browser.
- Gmail inbox, unread count, search, message preview, and sending.
- Discord webhook configuration stored through Orion Vault.
- Approval-gated email sending and Discord posting.
- Connect status and health commands.
- Home briefing card for unread email or connected services.

## Setup

Place a Google Desktop OAuth client file at:

`config/google-gmail-credentials.json`

Then run:

`connect add gmail`

For Discord, create a channel webhook and run:

`connect add discord`

## Safety

Orion always previews outbound email and Discord messages and requires explicit confirmation before sending.

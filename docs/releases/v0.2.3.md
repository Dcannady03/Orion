# Orion v0.2.3 — Pathfinder

## Theme

Orion can now safely explore its active workspace through a built-in Search
Plugin.

## Added

- Read-only `SearchSkill` service with workspace-boundary enforcement.
- `search <text>` content search with file and line reporting.
- `find <text>` command alias.
- `--files` file-name/path search.
- `--regex` regular-expression search.
- `--type <ext>` file-type filtering.
- `--path <path>` scoped workspace search.
- `--case-sensitive` matching.
- Conservative file-size and result limits.
- Generated-directory, binary-file, and oversized-file protection.
- Search plugin and skill unit tests.

## Repository Integrity

Removed an accidental nested copy of the Orion repository from inside the
`orion/` Python package, along with duplicated docs, configuration, tests, and
stale bytecode. The authoritative layout is now unambiguous.

## Quality Gate

All 37 unit tests pass before release packaging.

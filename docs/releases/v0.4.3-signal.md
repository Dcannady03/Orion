# Orion v0.4.3 — Signal

Signal introduces Orion's first two-way external interface. An approved Discord user can DM Orion or mention the bot in a server and receive a response from the same live Orion Brain used by the CLI. The interface is access-controlled, keeps normal server chatter out of Orion, and preserves the existing webhook notification channel.

## Setup

1. Create a Discord application and bot.
2. Enable Message Content Intent.
3. Invite the bot to a server.
4. Run `connect add discord bot`.
5. Start Orion with `python -m orion.main --discord`.

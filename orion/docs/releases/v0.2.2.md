# Orion v0.2.2 — Open Constellation

Released July 10, 2026.

## Milestone

Orion became extensible without requiring new capabilities to be hard-coded into its core.

## Added

- Plugin discovery from `plugins/*/plugin.py`
- Plugin lifecycle and controlled context
- Plugin command routing and dynamic help
- Plugin status and detailed information commands
- Failure isolation for broken plugins
- Code Skill migrated to the first Orion plugin
- Plugin development documentation

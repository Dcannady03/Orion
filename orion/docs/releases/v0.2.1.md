# Orion v0.2.1 — Project Memory

Project Memory is Orion's first persistent knowledge layer.

## Highlights
- Portable `.orion/` project metadata
- Persistent goals, phase, model preference, notes, and history
- Project-aware `history` and `about` commands
- Task storage foundation for the next milestone
- Constitution and professional release documentation

## Safety and ownership
Project data is plain JSON and Markdown inside the user's workspace. Writes are atomic, corrupt JSON is reported rather than silently erased, and Session Memory remains intentionally temporary.
